 Kurulum Talimatları
1. Sunucuyu Çalıştırma
- `ProfilesServer` klasörüne terminalden girin.
- `npm install` komutunu verin.
- `node server.js` komutunu verin.
- Sunucu varsayılan olarak `http://localhost:3000` adresinde çalışacaktır.

 2. Uygulamayı Çalıştırma
- `ProfilesApp` klasörüne gidin.
- `.env` dosyasını oluşturun ve bilgisayarınızın yerel IP adresini tanımlayın.
- `npm install` komutunu vverin.
- `npx expo start` komutunu verin.
- Telefonunuzdan Expo Go uygulaması ile QR kodu taratın.

 IP Konfigürasyonu
Uygulama, yerel sunucuya erişmek için aşağıdaki yapılandırmayı kullanmaktadır:
- API Base URL: http://192.168.1.105:3000
